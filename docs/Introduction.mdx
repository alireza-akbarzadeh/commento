# nextjs + TypeScript + Tailwind

**Project Overview:**

The it-bazar a market place store

**Technologies Stack:**

- **Nextjs Framework**: Empowers interactive user interface development.
- **Tailwind CSS**: Efficiently styles components with a utility-first approach.
- **TypeScript**: Enhances code reliability and scalability with static typing.
- **Zustand**: Provides lightweight state management for React apps.
- **ESLint & Prettier**: Ensures code quality and consistency.
- **PostCSS & Autoprefixer**: Enhances CSS styles for cross-browser compatibility.
- **Lucide React**: Offers a customizable set of icons for the application.
- **Radix UI**: Provides accessible and composable components.

**Development Practices:**

- **Linting**: Maintains code quality and adherence to best practices.
- **Type Checking**: Minimizes runtime errors with TypeScript's static typing.
- **Continuous Integration**: Integration with CI/CD pipelines enables automated testing and deployment.
- **Responsive Design**: Ensures compatibility across various devices and screen sizes.

"apple-beyond" offers a comprehensive learning journey for frontend developers, encompassing a diverse range of modern technologies and development methodologies.

**Project Starter Command:**

2. **Development:**

Starts the development server (`vite`).

```bash
   bun  dev
```

3. **Building:**

Builds the project with TypeScript and nextjs

```bash
   bun run build
```

4. **Linting and Formatting:**

Lints TypeScript and TypeScript JSX files using ESLint.

```bash
   bun run lint
```

Formats files using Prettier.

```bash
    npm run prettier
```

5. **Preview:**

Previews the production build (`vite preview`).

```bash
   bun run preview
```

6. **Storybook:**

Starts Storybook in development mode.

```bash
   bun run storybook
```

Builds Storybook for production deployment.

```bash
bun run build-storybook
```

7. **Git Hooks:**
   Husky is configured for code quality checks and pre-commit hooks.

```bash
bun prepare
```
